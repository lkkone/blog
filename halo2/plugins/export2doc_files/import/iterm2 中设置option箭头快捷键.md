# iterm2 中设置option箭头快捷键

### iterm2 中设置option箭头快捷键步骤

1. `cmd+,` 调出preference

2. `profiles -> keys`:

   keys

   

3. 分别找到 `⌥→`, `⌥ ←`, 设置两个参数的值:`Keyboard Shortcut` 与 `Action`:

   参数设置

   

4. `⌥→`设置为:

- Keyboard Shortcut: **Send Escape Sequence**

- Action:f

  右箭头

1. `⌥ ←` 设置为:

- Keyboard Shortcut: **Send Escape Sequence**

- Action:b

  左箭头
